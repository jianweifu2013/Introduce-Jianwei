Data Manipulation

  • dplyr   for data manipulation
  • data.table   for data manipulation
  • tidyr   for data tidying
  • stringr   for string manipulation
  • lubridate   for date and time manipulation


Visualization and Report

  • ggplot2   the most famous R package for making beautiful graphics
  • htmlwidgets   a simple way to build interactive visualization
  • shiny   build an interactive web application

Statistics
